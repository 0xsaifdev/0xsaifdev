### 🚀 About

I'm the **sole developer building an entire Hospital Management System** in Laravel — admissions, IPD, billing, the works. On the side I build indie apps and **sell the codebases**. I ship solo, lean hard on AI tooling, and document the process out loud.

* 🏥 **Day job:** Sole dev on a production HMS (Laravel) for a Pakistani healthcare org — currently building the IPD module end-to-end.
* 🧪 **Indie:** Building + selling MVPs and codebases. Calisto (calisthenics app) and more on the way.
* ⚡ **How I work:** Solo, AI-assisted (Claude Code), shipping in small finishable pieces.
* 🎓 **Education:** BS Software Engineering (graduating).
* 📍 **Based in** Hyderabad, Pakistan.
